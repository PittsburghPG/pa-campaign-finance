About this project

Price of the Prize collects and displays campaign finance information for major Pennsylvania races.

It was made by Alexandra Kanik, Eric Holmberg, Andrew McGill and Laura Malt Schneiderman.

Embed on your site

Copy and paste the code below to embed Price of the Prize on your website. (Your content management system must accept both Javascript code and iframes.) 

<div id="iframe" style="width:100%" ></div>
<script type="text/javascript" 
  src="//priceoftheprize.publicsource.org/js/pym.js"></script>
<script>
  var pymParent = new pym.Parent('iframe', 'http://priceoftheprize.publicsource.org/a/race', {});
  pymParent.onMessage("url-tracker", function(url){ 
  document.getElementById("iframe").removeChild(document.getElementById("iframe").firstChild)
    var pymParent = new pym.Parent('iframe', "http://priceoftheprize.publicsource.org" + url, {});
  });
</script>
